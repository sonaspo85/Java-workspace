Registering user profile settings

Create a user profile for a personalised service.

  • You can set up to two user profiles.
  • When you log in as guest, you cannot configure the profile settings. To set a different profile, log in with another user.

If no User profile has been created

  1. When a safety warning message appears, read it, select the driver image, and then press Confirm.

    The First Run Setup screen appears.

  2. Set the language to display on the screen and then press Next.
    • To use the default settings without changing the environment, press Skip.
  3. To set the driver's name, press Change name.
  4. To link to your Kia Connect account, press Link.

    The account linkage screen appears.

    • This feature can be set only for regions where Kia Connect is supported.
    • To use this feature, subscribe to Kia Connect. ▶ See "Subscribing to the service (if equipped)."
  5. Enter your phone number and authentication code, and then press Confirm.
    • When entering your phone number, make sure to include the + symbol and the country code.
  6. From among the backup files saved on the server, select the file(s) to load, and press Next.
  7. When the First Run Setup screen appears, press OK.

    Displays the Home screen.

    • To set the detailed user information and operation environment, press Profile settings. ▶ See "Configuring the user profile settings."

If a User profile has been created

  1. On the User Profile Settings screen, press Change user.
    • Available settings may differ depending on the vehicle model and specifications.
  2. Select the profile to use, and press Confirm.
  3. Set the language to display on the screen and then press Next.
    • To use the default settings without changing the environment, press Skip.
  4. To set the driver's name, press Change name.
  5. To link to your Kia Connect account, press Link.

    The account linkage screen appears.

    • This feature can be set only for regions where Kia Connect is supported.
  6. Enter your phone number and authentication code, and then press Confirm.
    • When entering your phone number, make sure to include the + symbol and the country code.
  7. From among the backup files saved on the server, select the file(s) to load, and press Next.
  8. When the First Run Setup screen appears, press OK.

    Displays the Home screen.

    • To set the detailed user information and operation environment, press Profile settings. ▶ See "Configuring the user profile settings."
