Open source information

This system contains open source software.

To obtain the source code developed under the open source licence installed on this product, please visit https://www.mobis.com/en/tech/rnd.do#open.

You can download all applicable licence notices, including the source code.

If you send an e-mail to MOBIS_OSSrequest@mobis.com within three years of purchasing this product to request an open source code for the software on this product, you will receive it in CD-ROM and other storage mediums with minimal charge, such as costs for medium and transportation.
