  • The following files are not played.
    • Modified files (format-changed files)
    • Variant files (e.g. MP3 files encoded by WMA audio codec)
    • DRM (Digital Rights Management)-protected files
    • Files with no index
  • Sound quality is not guaranteed in case of music files that use the bit-rate exceeding 192 kbps.
  • With files that do not use a fixed bit rate, some functions may not work properly or the playback time may not appear accurately.
  • If an issue occurs while playing or you want to play an incompatible format file, use the latest encoder software to change the format before using the file.
