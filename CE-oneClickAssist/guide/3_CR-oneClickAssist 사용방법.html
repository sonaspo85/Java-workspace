16시01분34초 - 목차

>> 3_CR-oneClickAssist 사용방법

1. CR-oneClickAssist란

  • HTML 문서에서 이미지 클릭시, Excel 시트에 정리되어 있는 URL 주소로 이동되도록 href 속성을 추가 해주는 프로그램이다.
  • oneclick 프로그램으로 추출한 HTML 결과물을 소스 파일로 활용 한다.

2. 사용 방법

1. oneClick-Assist.jar 실행 하여, 'QR code URL' 클릭 -> URL가 정리된 Excel 선택

1. 유의 사항
  • 엑셀 파일은 HTML 경로내에 배치 되어야 한다.
  • 엑셀 시트는 하기 구조로 유지 되어야 한다.

2. 1번의 다이얼로그 창에서 Excel 선택 후 OK 클릭시 프로세스 작동이 진행되며, 작업 완료시, 완료 알림창 출력

3. 브라우저로 HTML문서를 열어 URL 주소가 제대로 삽입되었는지 확인
