시스템 살펴보기

시스템 켜기/끄기

시스템을 켜거나 끄는 방법을 설명합니다. 시스템을 사용하기 전에 반드시 사용설명서를 읽고 숙지하여, 시스템을 안전하고 유용하게 사용하세요.

시스템 켜기

  1. 차량의 시동을 거세요.
  2. 안전 경고문이 나타나면 내용을 읽은 후 동의함을 누르세요.
    • 사용 언어를 변경하려면 언어/Language를 누르세요.
  • 사용자 프로필을 설정하려면 다음을 참고하세요. ▶  '사용자 초기 설정하기' 참고

시스템 끄기

차량 운행 중 시스템을 끄려면 조작 패널에서 전원 버튼을 길게 누르세요.

화면과 소리가 꺼집니다.

  • 시스템을 다시 사용하려면 전원 버튼을 누르세요.
  • 차량의 시동을 끈 후 일정 시간이 지나면 시스템이 자동으로 꺼지며, 운전석 문을 열면 시스템이 바로 꺼집니다.

경고

  • •  안전을 위해, 주행 중에는 영상 화면이 보이지 않습니다. 반드시 정차한 후에 영상을 시청하세요. 수동 변속기 차량의 경우 주차 브레이크를 건 후에, 자동 변속기 차량의 경우 'P'(주차)로 변속하거나 주차브레이크를 건 후에 영상을 시청할 수 있습니다.
  • •  영상 시청 외에도 안전을 위해, 주행 중에는 실행되지 않는 기능이 있습니다. 이런 기능들은 정차한 상태에서만 정상적으로 작동합니다. 정차한 후에 기능을 사용하세요.
  • •  화면이 보이지 않거나 소리가 나지 않는 등 시스템이 고장 난 경우 사용을 중단하세요. 고장 난 상태로 계속 사용하면 화재, 감전 또는 치명적인 고장의 원인이 됩니다.

주의

  • •  시동 스위치가 'ACC' 또는 'ON' 위치에 있을 때에도 시스템 전원을 켤 수 있지만, 시동을 걸지 않고 시스템을 장시간 사용하면 배터리가 방전될 수 있습니다. 시스템을 장시간 사용하려면 차량의 시동을 거세요.
  • •  시동이 꺼진 상태로 시스템을 사용하면 배터리 경고 메시지가 나타납니다. 시동을 걸면 배터리 경고 메시지가 사라집니다.
  • 시동을 껐다 켠 경우 시동을 끌 때 사용하던 모드와 설정대로 시스템이 켜집니다.
  • 시동을 걸 때 갑자기 큰소리로 오디오가 작동하는 것을 방지하려면 시동을 끄기 전에 시스템 음량을 적절하게 조절하세요.
  • 시스템의 음량이 자동으로 작아지게 설정하려면 전체 메뉴 화면에서 설정 ▶ 사운드 ▶ 고급 음향 ▶ 오디오 음량 자동 줄이기를 눌러 기능을 설정하세요.
