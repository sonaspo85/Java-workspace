유용한 기능

폰 프로젝션 사용하기

USB 케이블을 사용해 차량의 USB 단자와 스마트폰을 연결하면 시스템의 넓은 화면과 편리한 사용자 인터페이스를 이용해 Android 스마트폰이나 iPhone의 앱을 실행하고 제어할 수 있습니다.

주의

휴대 전화 제조사에서 제공하는 정품 USB 케이블 사용을 권장합니다. 정품이 아닌 USB 케이블을 사용할 경우 기능이 올바르게 동작하지 않거나 시스템이 고장 날 수 있습니다.

Android Auto로 Android 스마트폰과 연동하기

Android Auto를 사용해 Android 스마트폰과 시스템을 연동하면 시스템 화면에서 Android 스마트폰의 앱을 실행하고 제어할 수 있습니다.

  1. Android Auto를 사용하려면 먼저 다음을 확인하세요.
    • •  Android Auto 홈페이지(https://www.android.com/auto)에서 Android Auto를 지원하는 국가, 기기 종류, 모델을 확인하세요.
    • •  Google Play 스토어에서 Android Auto 앱을 다운로드해 설치하세요. 미지원 국가에서는 Android Auto를 설치할 수 없습니다.
  2. 전체 메뉴 화면에서 설정 ▶ 기기 연결 ▶ 폰 프로젝션 설정 ▶ Android Auto를 누른 후 Android Auto에 체크하세요.
    • 분할 화면 사용을 체크하면 Android Auto 사용 시 분할 화면이 표시됩니다.
  3. 스마트폰 구입 시 제공된 USB 케이블을 사용해 스마트폰을 차량의 USB 단자에 연결하세요.
    • 처음 연결하면 시스템 화면에 알림창이 나타납니다. 내용을 확인한 후 확인을 누르세요.
    • Android Auto를 종료하고 스마트폰 연동을 해제하려면 시스템에서 USB 케이블을 분리하세요.

    주의

    USB 커넥터를 짧은 시간에 연결하고 분리하는 동작을 반복하지 마세요. 기기 동작에 오류가 발생할 수 있으며 시스템이 고장 날 수 있습니다.

  4. 스마트폰이 잠겨 있다면 잠금을 해제한 후 Android Auto 사용에 동의하세요.

    스마트폰이 정상적으로 인식되면 전체 메뉴 화면에 Android Auto 메뉴가 활성화됩니다.

Android Auto 사용하기

  1. 전체 메뉴 화면에서 Android Auto를 누르세요.
  2. 다음 중 원하는 기능을 실행해 사용하세요.
    • •  스마트폰의 전화, 음악, 내비게이션 기능을 실행하려면 해당 앱의 아이콘을 누르세요.
      • 폰 프로젝션으로 내비게이션 앱을 사용하면 시스템의 방송/미디어와 내비게이션 앱의 오디오가 함께 출력됩니다. 두 오디오가 함께 출력될 때 음량 노브를 돌리면 내비게이션 앱의 음량이 우선적으로 조절됩니다.
    • •  Google 음성인식 기능을 실행하려면 스티어링 휠에서 음성인식 버튼을 길게 누르세요.
  • 스티어링 휠에서 [MODE] 버튼을 누르면 시스템의 방송/미디어 모드와 함께 스마트폰에 설치된 미디어 앱도 실행할 수 있습니다.
  • 스티어링 휠에서 [MODE] 버튼을 눌러 실행할 기능을 설정하려면, [MODE] 버튼을 길게 눌러 설정 화면으로 이동하세요.
  • 스티어링 휠에서 통화 버튼을 누르면 블루투스 전화가 실행됩니다.
  • 폰 프로젝션이 실행 중인 경우 폰 프로젝션 설정을 변경할 수 없습니다. 설정을 변경하려면 USB 케이블을 분리하세요.
  • 화면 아이콘은 Google사의 정책에 따라 변경될 수 있습니다.
  • Android Auto로 사용할 수 있는 앱이나 기능은 Google사의 정책에 따라 변경될 수 있습니다. 지원되는 기능을 확인하려면 Android 홈페이지(https://www.android.com/auto)를 방문하세요.
  • 최적화된 환경에서 사용하려면 연결하려는 스마트폰의 OS를 최신 버전으로 업데이트하세요.
  • 연결된 기기의 종류, 펌웨어, OS 버전에 따라 인식하는 시간에 차이가 있을 수 있습니다.
  • Android Auto 사용 시 통신 요금이 부과될 수 있습니다.
  • 네트워크 신호 상태에 따라 일부 기능이 원활하게 작동하지 않을 수 있습니다.
  • Android Auto 사용 중에는 스마트폰 조작을 자제하세요.
  • Android Auto 사용 중에는 다음 기능을 사용할 수 없습니다.
    • USB 모드
    • Apple CarPlay
    • 조작 패널 또는 스티어링 휠 리모컨을 이용한 탐색 기능

Apple CarPlay로 iPhone과 연동하기

Apple CarPlay를 사용해 iPhone과 시스템을 연동하면 시스템 화면에서 iPhone의 앱을 실행하고 제어할 수 있습니다.

  1. Apple CarPlay를 사용하려면 먼저 다음을 확인하세요.
    • •  Apple 홈페이지(https://www.apple.com/kr/ios/carplay)에서 Apple CarPlay를 지원하는 iPhone 모델을 확인하세요.
    • •  iPhone의 설정에서 Apple CarPlay와 Siri 기능이 켜져 있는지 확인하세요. Apple CarPlay 또는 Siri 기능이 없는 경우, 최신 iOS 버전으로 업데이트하세요.
  2. 전체 메뉴 화면에서 설정 ▶ 기기 연결 ▶ 폰 프로젝션 설정 ▶ Apple CarPlay를 누른 후 Apple CarPlay에 체크하세요.
    • 분할 화면 사용을 체크하면 Apple CarPlay 사용 시 분할 화면이 표시됩니다.
  3. iPhone 구입 시 제공된 USB 케이블을 사용해 iPhone을 차량의 USB 단자에 연결하세요.
    • 처음 연결하면 시스템 화면에 알림창이 나타납니다. 내용을 확인한 후 확인을 누르세요.
    • Apple CarPlay를 종료하고 iPhone 연동을 해제하려면 시스템에서 USB 케이블을 분리하세요.

    주의

    USB 커넥터를 짧은 시간에 연결하고 분리하는 동작을 반복하지 마세요. 기기 동작에 오류가 발생할 수 있으며 시스템이 고장 날 수 있습니다.

  4. iPhone이 잠겨 있다면 잠금을 해제한 후 Apple CarPlay 사용에 동의하세요.

    iPhone이 정상적으로 인식되면 전체 메뉴 화면에 Apple CarPlay 메뉴가 활성화됩니다.

Apple CarPlay 사용하기
