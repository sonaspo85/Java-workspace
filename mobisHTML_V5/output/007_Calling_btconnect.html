블루투스 기기 연결하기

블루투스(Bluetooth)는 근거리 무선 네트워킹 기술입니다. 블루투스 통신 기능을 사용하면 케이블 연결 없이 가까운 거리에 있는 블루투스 기기를 서로 연결하여 데이터를 주고받을 수 있어, 각종 기기를 보다 효율적으로 사용할 수 있습니다.

블루투스 기기를 사용해 전화를 걸거나 오디오를 재생하려면 먼저 다음을 확인하세요.

  • •  시스템에 등록되어 있지 않은 기기의 경우 다음을 실행하세요. ▶  '기기 등록하기' 참고
  • •  시스템에 등록된 기기의 경우 기기를 시스템에 가까이 가져가면 자동으로 연결됩니다. 자동으로 연결되지 않으면 다음을 실행하세요. ▶  '등록된 기기 연결하기', '등록된 기기 연결 해제하기' 참고

경고

안전한 곳에 정차한 후 블루투스 기기를 연결하세요. 주행 중에 연결을 시도하다가 전방 주시 태만이나 부주의로 인해 사고가 발생하면 운전자 또는 탑승자가 사망하거나 부상을 입을 수 있습니다.

  • 블루투스 연결 기능은 핸즈프리 및 오디오 관련 기능만을 지원합니다. 휴대 전화나 MP3 플레이어 등, 블루투스 핸즈프리 또는 오디오 기능을 지원하는 기기를 연결해 사용하세요.
  • 기능 호환성에 대한 정보를 보려면 제네시스 홈페이지(http://www.genesis.com)를 방문하여 블루투스 호환 안내를 참조하세요.

기기 등록하기

블루투스 기기를 시스템에 연결하려면 먼저 해당 기기를 시스템의 블루투스 기기 목록에 등록해야 합니다.

  1. 전체 메뉴 화면에서 설정 ▶ 기기 연결 ▶ 기기 연결 ▶ 신규 추가를 누르세요.
    • 블루투스 기기를 처음 등록한다면 스티어링 휠에서 통화 버튼을 누르거나 전체 메뉴 화면에서 전화를 눌러도 됩니다.
  2. 연결할 블루투스 기기에서 블루투스를 실행한 후 검색된 기기 목록에서 차량의 시스템을 찾아 선택하세요.
    • 시스템의 블루투스 이름은 신규 기기 추가 알림창에서 확인하세요.
  3. 블루투스 기기 화면과 시스템 화면의 인증 번호가 동일한지 확인한 후 기기에서 연결을 승인하세요.
  4. 블루투스 기기가 휴대 전화인 경우, 휴대 전화의 연락처를 시스템으로 다운로드하거나 문자 메시지 수신 알림을 차량에서 사용하려면 휴대 전화에서 접근 요청을 허용하세요.
    • 접근 권한은 블루투스 전화 기능을 사용할 때만 필요하므로 오디오 기기를 연결하는 경우 이 과정이 나타나지 않습니다.
  • 휴대 전화에서 접근 요청을 허용한 후 블루투스 연결이 완료될 때까지 시간이 걸릴 수 있습니다. 블루투스 연결이 완료되면 블루투스 연결 상태 표시 아이콘이 화면 상단에 표시됩니다.
  • 데이터에 대한 접근 요청 허용 설정은 휴대 전화의 블루투스 설정 메뉴를 이용해 변경할 수 있습니다. 자세한 내용은 휴대 전화 사용설명서를 참고하세요.
  • 블루투스 자동 연결 기능을 사용하지 않으려면 기기의 블루투스 기능을 끄세요.
  • 블루투스 기기는 최대 6개까지 등록할 수 있습니다.

등록된 기기 연결하기

블루투스 기기를 사용하려면 등록된 기기를 시스템에 연결해야 합니다.

  • 블루투스 핸즈프리 1개 또는 블루투스 오디오 기기 2개를 연결할 수 있습니다.
  1. 전체 메뉴 화면에서 설정 ▶ 기기 연결 ▶ 기기 연결을 누르세요.
  2. 연결할 기기의 아이콘을 누르세요.
  • 블루투스 기기가 연결되지 않을 경우 연결할 블루투스 기기의 블루투스 기능이 켜져 있는지 확인하세요.
  • 연결된 기기가 연결 범위를 벗어나거나 기기 이상으로 연결이 비정상적으로 종료된 경우, 기기가 연결 범위 안으로 다시 돌아오거나 정상 상태가 되면 연결이 자동으로 복원됩니다.

블루투스 연결 화면

  1. 이전 단계 화면으로 이동합니다.
  2. 검색어를 입력하거나 음성으로 설정 항목을 검색할 수 있습니다.
  3. 핸즈프리 또는 블루투스 오디오 중 사용할 기능을 선택하여 블루투스 기기를 연결하거나 해제할 수 있습니다.
  4. 기기를 새로 등록할 수 있습니다.
  5. 등록된 기기를 삭제할 수 있습니다.
  6. 관련된 메뉴 목록이 표시됩니다.
    • •  화면 끄기: 화면을 끌 수 있습니다. 화면을 다시 켜려면 화면을 누르거나 전원 버튼을 짧게 누르세요.
    • •  설정 검색: 검색어를 입력하거나 음성으로 설정 항목을 검색할 수 있습니다.
    • •  웹 매뉴얼 > 기기 연결 설정: 시스템의 웹 매뉴얼로 연결 가능한 QR 코드를 확인할 수 있습니다.

등록된 기기 연결 해제하기

  1. 전체 메뉴 화면에서 설정 ▶ 기기 연결 ▶ 기기 연결을 누르세요.
  2. 연결된 기기의 아이콘을 누르세요.

등록된 기기 삭제하기

블루투스 기기를 더 이상 사용하지 않거나, 6개의 기기가 이미 등록되어 있을 때 새로운 기기를 연결하려면 등록된 기기를 삭제하세요.

  1. 전체 메뉴 화면에서 설정 ▶ 기기 연결 ▶ 기기 연결 ▶ 기기 삭제를 누르세요.
  2. 삭제할 기기를 선택한 후 삭제를 누르세요.

    등록된 기기를 모두 삭제하려면 전체 선택 ▶ 삭제를 누르세요.

  3. 예를 누르세요.
  • 등록된 기기를 삭제하면 시스템에 다운로드된 연락처, 통화 목록 및 문자 메시지함의 내용이 모두 삭제됩니다.
