증상별 원인과 해결 방법

시스템에 이상이 생기면 먼저 다음 표를 참고해 이상 유무를 확인하고 직접 문제를 해결해 보세요.

설명된 방법대로 조치해도 계속 문제가 해결되지 않으면 자사 서비스 센터 또는 서비스 협력점에 문의하세요.

소리와 영상 관련

증상

예상되는 원인

해결 방법

소리와 영상이 나오지 않습니다.

시스템 꺼짐

  • •  시동 스위치가 'ACC' 또는 'ON' 에 놓여 있는지 확인하세요.
  • •  조작 패널에서 전원 버튼을 눌러 시스템을 켜세요.

음량 작음

조작 패널에서 음량 노브를 돌려 음량을 조절하세요.

음소거 실행

스티어링 휠에서 음소거 버튼을 눌러 음소거를 해제하세요.

한쪽 스피커에서만 소리가 납니다.

사운드 출력 불균형

전체 메뉴 화면에서 설정 ▶ 사운드로 진입해 설정 메뉴에서 소리가 집중되는 위치를 적절하게 조절하세요.

소리가 끊기거나 잡음이 들립니다.

시스템 진동

시스템에 진동이 발생하면 소리가 끊기거나 잡음이 생길 수 있습니다. 이는 제품의 고장이 아니며, 진동이 멈추면 시스템이 정상적으로 작동합니다.

영상이나 이미지의 화질이 떨어집니다.

화면 조정 필요

전체 메뉴 화면에서 설정 ▶ 디스플레이로 진입해 설정 메뉴에서 화면을 조정하세요.

시스템 진동

시스템에 진동이 발생하면 이미지가 왜곡될 수 있습니다. 이는 제품의 고장이 아니며, 진동이 멈추면 시스템이 정상적으로 작동합니다.

디스플레이의 노화 또는 성능 저하

같은 현상이 반복되면 자사 서비스 센터 또는 서비스 협력점에 문의하세요.

화면에 빨간색, 파란색, 녹색의 작은 점이 있습니다.

LCD는 대단히 점밀도가 높은 기술로 만들어져 있으므로 전체 화소수의 0.01 % 이하에서 화소 부족(특정 화소가 항상 꺼져 있는 픽셀) 또는 항상 점등(특정 화소가 항상 켜져 있는 픽셀)이 있을 수 있습니다.

USB 재생 관련

증상

예상되는 원인

해결 방법

USB 저장 장치의 파일이 재생되지 않습니다.

재생할 수 있는 파일 없음

USB 저장 장치에 시스템에서 재생할 수 있는 파일을 저장한 후 다시 연결하세요. ▶  'USB 모드' 참고

연결 상태 불량

USB 저장 장치를 차량의 USB 단자에서 빼냈다가 다시 바르게 연결하세요.

USB 저장 장치 오염

USB 저장 장치와 차량의 USB 단자의 접촉면에서 이물질을 제거하세요.

USB 허브 또는 연장 케이블 사용

USB 저장 장치를 차량의 USB 단자에 직접 연결해 사용하세요.

비규격품 USB 저장 장치 사용

시스템에서 재생할 수 있는 USB 저장 장치를 사용하세요. ▶  'USB 모드' 참고

USB 저장 장치 손상

PC에서 USB 저장 장치를 포맷한 후 다시 연결하세요. 포맷할 때에는 FAT16/32 방식을 선택하세요.

블루투스 연결 관련

증상

예상되는 원인

해결 방법

블루투스 기기에서 시스템이 검색되지 않습니다.

등록 모드 실행 필요

전체 메뉴 화면에서 설정 ▶ 기기 연결로 진입해 기기 등록 모드를 실행한 후 기기에서 다시 검색하세요. ▶  '블루투스 기기 연결하기' 참고

블루투스 기기가 연결되지 않습니다.

블루투스 기능 실행 필요

연결할 블루투스 기기에서 블루투스 기능을 켜세요.

블루투스 기능 오류

  • •  블루투스 기기의 블루투스 기능을 껐다가 켠 후 다시 연결하세요.
  • •  블루투스 기기를 껐다가 켠 후 다시 연결하세요.
  • •  블루투스 기기의 배터리를 분리했다가 장착하고 전원을 켠 후 다시 연결하세요.
  • •  시스템과 블루투스 기기 양쪽에서 등록을 해제한 후 다시 등록해 연결하세요.

폰 프로젝션 관련

증상

예상되는 원인

해결 방법

USB 케이블로 스마트폰 연결 시 폰 프로젝션 기능이 실행되지 않습니다.

폰 프로젝션 미지원

기능별 홈페이지에서 연결하려는 스마트폰이 폰 프로젝션 기능을 지원하는지 확인하세요.

  • •  Android Auto: https://support.google.com/androidauto
  • •  Apple CarPlay: https://www.apple.com/kr/ios/carplay

기능 비활성화

  • •  전체 메뉴 화면에서 설정 ▶ 기기 연결로 진입해 폰 프로젝션 설정 메뉴에서 연결 기능을 활성화 하세요.
  • •  스마트폰의 앱 설정 또는 차단 설정에서 폰 프로젝션 기능이 차단되지 않았는지 확인하세요.

스마트폰 상태 부적합 또는 동작 이상

  • •  스마트폰의 배터리 잔량이 부족하지 않은지 확인하세요. 배터리 잔량이 부족하면 인식이 잘 되지 않을 수 있습니다.
  • •  네트워크 신호 세기가 약하면 폰 프로젝션 기능이 정상적으로 동작하지 않을 수 있습니다.
  • •  스마트폰이 잠겨 있다면 잠금을 해제하세요.
  • •  스마트폰을 초기화한 후 다시 연결하세요.

폰 프로젝션 기능 실행 시 또는 사용 중에 검은 화면이 표시됩니다.

스마트폰 동작 이상

  • •  스마트폰에서 USB 케이블을 분리했다가 다시 연결하세요.
  • •  스마트폰을 초기화한 후 다시 연결하세요.

시스템 작동 관련

증상

예상되는 원인

해결 방법

시스템을 껐다가 다시 켰을 때 이전에 사용한 미디어 모드가 표시되지 않습니다.

미디어 연결 불량 또는 재생 오류

해당 미디어가 연결되지 않거나 정상적인 재생 상태가 아니라면 그 이전에 사용했던 모드가 표시됩니다. 미디어를 다시 연결하거나 다시 재생하세요.

시스템이 느려지거나 동작하지 않습니다.

시스템 내부 오류

  • •  펜이나 클립과 같이 끝이 뾰족한 도구로 리셋 버튼을 길게 눌러 시스템을 다시 시작하세요.
  • •  시스템 소프트웨어를 최신 버전으로 업데이트하세요. ▶  내비게이션 업데이트 사이트, '무선 업데이트'참고
  • •  같은 현상이 반복되면 자사 서비스 센터 또는 서비스 협력점에 문의하세요.

시스템이 켜지지 않습니다.

퓨즈 문제

  • •  정비소를 방문해 적절한 용량의 퓨즈로 교체하세요.
  • •  같은 현상이 반복되면 자사 서비스 센터 또는 서비스 협력점에 문의하세요.

증상

예상되는 원인

해결 방법

목적지가 검색되지 않습니다.

최신 지도 데이터 미반영

지도를 최신 버전으로 업데이트하세요. ▶  내비게이션 업데이트 사이트, '무선 업데이트'참고

길 안내가 실제 도로와 일치하지 않습니다.

최신 지도 데이터 미반영

지도를 최신 버전으로 업데이트하세요. ▶  내비게이션 업데이트 사이트, '무선 업데이트'참고

지도 업데이트 이후 도로 변경

지도를 최신 버전으로 업데이트한 후에도 도로와 표지판 등이 수시로 변경될 수 있으므로 길 안내가 실제 도로와 다를 수 있습니다. 실제 도로 상황과 교통 규제에 따라 주행하세요.

길 안내 중 경로를 우회해 안내하거나 단속 카메라를 안내하지 않습니다.

최신 시스템 데이터 및 지도 데이터 미반영

시스템 및 지도를 최신 버전으로 업데이트하세요. ▶  내비게이션 업데이트 사이트, '무선 업데이트'참고

실시간 도로 교통 정보(TPEG)가 수신되지 않습니다.

전파 신호 세기 약함

  • •  일부 산간 지역 및 도서 지역에서는 교통 정보 수신이 원활하지 않을 수 있습니다. 전파 수신이 원활한 장소에서 수신 상태를 확인하세요.
  • •  같은 현상이 반복되면 자사 서비스 센터 또는 서비스 협력점에 문의하세요.
