시스템 살펴보기

각 부분의 이름과 기능

조작 패널 및 스티어링 휠의 각 부분의 명칭과 조작 기능을 설명합니다.

조작 패널

  • 차종 및 사양에 따라 형상이 다를 수 있습니다. 자세한 내용은 별도로 제공되는 '간편 사용설명서'를 참고하세요.

전원 버튼
(PWR)/
음량 노브
(VOL)

  • •  방송/미디어 기능 켜기 또는 끄기
  • •  길게 누르면 화면 및 소리 끄기
  • •  돌리면 시스템 음량 조절(내비게이션 음량 제외)

MAP 버튼

  • •  현재 위치의 지도 화면으로 이동
  • •  지도 화면에서 길 안내 중에 누르면 안내 음성 재출력

NAV 버튼

  • •  내비게이션 메뉴 화면으로 이동
  • •  길게 누르면 내비게이션 검색 화면으로 이동 ▶  '통합 검색 사용하기' 참고

RADIO 버튼

  • •  방송 실행
  • •  방송 실행 중 반복해서 누르면 FM/AM/DMB 모드 변경
  • •  길게 누르면 방송/미디어 선택 창 실행

MEDIA 버튼

  • •  연결된 미디어 실행
  • •  길게 누르면 방송/미디어 선택 창 실행

사용자 버튼
()

  • •  사용자 정의 기능 실행
  • •  길게 누르면 기능 설정 화면으로 이동

SETUP 버튼

  • •  설정 화면으로 이동
  • •  길게 누르면 버전 정보 화면으로 이동

탐색 노브
(TUNE/FILE)

  • •  라디오 청취 중 주파수 조정/방송 채널 변경
  • •  DMB 시청 중 채널 탐색
  • •  미디어 재생 중 곡/파일 탐색
  • •  탐색 중 현재 채널/곡/파일 선택
  • •  지도 화면에서 지도 확대/축소(해당 기능 설정 시)

시스템 리셋 버튼

  • •  시스템 다시 시작

스티어링 휠 리모컨

  • 차종 및 사양에 따라 형상이 다를 수 있습니다. 자세한 내용은 별도로 제공되는 '간편 사용설명서'를 참고하세요.

음성인식 버튼
()

  • •  누르면 음성인식 시작. 길게 누르면 음성인식 종료
  • •  길게 누르면 폰 프로젝션으로 연결된 스마트폰의 음성인식 시작 또는 종료(스마트폰 사양에 따라 버튼의 동작이 다를 수 있음)

MODE 버튼

  • •  반복해서 누르면 시스템 모드(라디오, 미디어 등) 변경
  • •  길게 누르면 기능 설정 화면으로 이동

음량 레버
(+/-)

  • •  음량 조절

음소거 버튼
()

  • •  누르면 음소거 실행 또는 해제(내비게이션 음량 제외)
  • •  미디어 재생 중 누르면 일시 정지 또는 다시 시작

이동 레버
(/)

  • •  라디오 청취 중 프리셋 채널 변경. 길게 누르면 방송 탐색
  • •  DMB 시청 중 방송 채널 변경
  • •  미디어 재생 중 곡/파일 변경. 길게 누르면 되감기 또는 빨리 감기(블루투스 오디오 모드 제외)

통화 버튼
()

  • •  블루투스 연결 전: 누르면 블루투스 연결 창 표시
  • •  블루투스 연결 후: 누르면 최근 통화 목록 표시, 길게 누르면 최근 통화 번호로 재다이얼
  • •  전화 수신 시: 누르면 전화 받기

통화 종료 버튼
()

  • •  블루투스 통화 중: 누르면 통화 종료
  • •  전화 수신 시: 길게 누르면 전화 거절

사용자 버튼
()

  • •  사용자 정의 기능 실행
  • •  길게 누르면 기능 설정 화면으로 이동
