내비게이션

다양한 방법으로 길 찾기

검색/경로 메뉴에서는 위치 정보를 직접 입력해 검색하거나 카테고리별로 분류된 장소 또는 최근 사용한 목적지를 선택하는 등, 보다 다양한 검색 방법을 이용해 목적지를 설정할 수 있습니다.

다음 중 원하는 방법을 선택해 실행하세요.

  • •  전체 메뉴 화면에서 검색/경로를 누르세요.
  • •  조작 패널에서 [NAV] 버튼을 누르세요.

통합 검색 사용하기

이름, 주소, 전화번호, 시설명, 검색어 등 다양한 목적지 정보를 입력한 후 검색 결과에서 목적지를 찾아 설정할 수 있습니다.

  1. 검색/경로 화면에서 통합 검색을 누르세요.
  2. 이름, 전화번호, 주소 등의 목적지 정보를 입력하세요.

    입력한 내용에 맞는 추천어가 화면 왼쪽에 실시간으로 표시됩니다.

    • 다음의 예를 참고하세요.
        예) 현대자동차(본사), 02-3464-0000, 서울특별시 서초구 헌릉로 12
    • 초성만 입력해도 검색할 수 있습니다.
        예) 현대기아차 본사: ㅎㄷㄱㅇㅊㅂㅅ
    • 검색 결과를 보려면 추천어를 선택하거나 확인을 누르세요.
    • 통합 검색 화면에서 시스템의 웹 매뉴얼로 연결하려면 ▶ 웹 매뉴얼을 누른 후 QR 코드를 스캔하세요.
  3. 검색 결과 목록에서 원하는 목적지를 선택하세요.
    • 검색 결과를 특정 지역으로 간추려 보려면 전국을 누른 후 원하는 지역을 선택하세요.
    • 검색 결과 목록의 정렬 순서를 변경하려면 정렬 기준을 눌러 원하는 항목을 선택하세요.
  4. 지도상에 표시된 목적지의 위치를 확인한 후 목적지 설정을 누르세요.
    • 화면 오른쪽 하단의 주차장을 누르면 지도 화면에 주차장 아이콘이 표시됩니다.
  5. 탐색된 경로 목록에서 원하는 경로를 선택한 후 안내 시작을 누르세요.

주소로 검색하기

  1. 검색/경로 화면에서 주소 검색을 누르세요.
  2. 주소 유형(도로명 주소 또는 지번 주소)을 선택하세요.
    • 도로명 주소의 예: 서울특별시 서초구 헌릉로 12
    • 지번 주소의 예: 서울특별시 서초구 양재동 231
    • 주소 검색 화면에서 시스템의 웹 매뉴얼로 연결하려면 ▶ 웹 매뉴얼을 누른 후 QR 코드를 스캔하세요.
  3. 지역과 도로 이름 또는 동 이름을 차례로 선택하세요.
    • 도로 이름이나 동 이름을 직접 입력해 주소를 검색하려면 화면 오른쪽 상단에서 도로명 검색 또는 동이름 검색을 누르세요.
  4. 화면의 안내에 따라 상세 주소를 입력한 후 확인을 누르세요.
  5. 지도상에 표시된 목적지의 위치를 확인한 후 목적지 설정을 누르세요.

주변 시설 검색하기

카테고리별로 주변 시설을 검색해 목적지를 설정할 수 있습니다.

  1. 검색/경로 화면에서 주변 검색을 누르세요.
  2. 화면의 안내에 따라 세부 시설을 선택하세요.
    • 주변 검색 화면에서 시스템의 웹 매뉴얼로 연결하려면 ▶ 웹 매뉴얼을 누른 후 QR 코드를 스캔하세요.
  3. 검색 결과 목록에서 목적지를 선택하세요.
    • 검색 결과를 분류해서 표시하려면 화면 왼쪽에서 분류 기준을 선택하세요.
    • 검색 결과 목록의 정렬 순서를 변경하려면 정렬 기준을 눌러 원하는 항목을 선택하세요.
  4. 지도상에 표시된 목적지의 위치를 확인한 후 목적지 설정을 누르세요.
    • 지도에 표시되는 주변 시설 종류를 변경하려면 지도 화면에서 ▶ 주변 정보 ▶ 주변 시설을 누르세요.
    • 길 안내 중에 현재 위치의 주변 시설을 직접 입력해 검색하려면 지도화면에서 ▶ 주변 정보 ▶ 주변 검색을 누르세요.

최근 목적지에서 검색하기

최근에 경로를 탐색한 장소 중에서 원하는 곳을 선택해 목적지로 설정할 수 있습니다.

  1. 검색/경로 화면에서 최근 목적지를 누르세요.

    최근에 경로를 탐색한 목적지 목록이 나타납니다.

  2. 목록에서 원하는 장소를 선택하세요.
    • 최근 목적지 화면에서 시스템의 웹 매뉴얼로 연결하려면 ▶ 웹 매뉴얼을 누른 후 QR 코드를 스캔하세요.
  3. 지도상에 표시된 목적지의 위치를 확인한 후 목적지 설정을 누르세요.

최근 목적지 삭제하기

  1. 최근 목적지 화면에서 삭제를 누르세요.
  2. 삭제할 목적지에 체크한 후 삭제를 누르세요.

    선택한 장소 또는 모든 장소를 삭제할 수 있습니다.

정비소 검색하기

서비스 센터나 정비소의 위치를 쉽게 검색할 수 있습니다.

  1. 검색/경로 화면에서 정비소 검색을 누르세요.
  2. 검색 결과 목록에서 정비소나 서비스 센터를 선택하세요.
    • 검색 결과 목록의 정렬 순서를 변경하려면 정렬 기준을 눌러 원하는 항목을 선택하세요.
    • 정비소 검색 화면에서 시스템의 웹 매뉴얼로 연결하려면 ▶ 웹 매뉴얼을 누른 후 QR 코드를 스캔하세요.
  3. 지도상에 표시된 목적지의 위치를 확인한 후 목적지 설정을 누르세요.

경로 편집하기

목적지로 가는 길에 다른 곳을 경유하도록 경로를 편집할 수 있습니다. 경유지는 두 곳까지 추가할 수 있습니다.

경유지 설정하기

원하는 장소를 검색한 후 경유지로 설정할 수 있습니다.

  1. 검색/경로 화면에서 원하는 검색 방법을 눌러 경유지를 검색하세요.
    • 경유지 검색 방법에 대한 자세한 내용은 다음을 참고하세요.
        ▶ '통합 검색 사용하기', '주소로 검색하기', '주변 시설 검색하기', '최근 목적지에서 검색하기', '정비소 검색하기' 참고
  2. 검색한 경유지의 지도 화면에서 경유지 추가를 누르세요.

    해당 위치가 경로에 바로 추가됩니다.

    • 차종 및 사양에 따라 표시되는 화면이 다를 수 있습니다.
  3. 경유지 설정이 완료되면 경로 탐색을 누르세요.
  4. 재탐색된 경로를 선택한 후 안내 시작을 누르세요.

경유 순서 변경하기

설정된 경로를 확인하고 경유 순서를 변경할 수 있습니다.

  1. 검색/경로 화면에서 경로 편집을 누르세요.
  2. 경유 순서를 변경할 장소를 선택한 후 순서 변경을 누르세요.
    • 경유 순서를 변경하려면 순서 변경을 누른 후 원하는 위치로 드래그하세요.
    • 경유지를 추가로 설정하려면 경로 편집 화면에서 경유지 추가를 누르세요.
    • 경로 편집 화면에서 시스템의 웹 매뉴얼로 연결하려면 ▶ 웹 매뉴얼을 누른 후 QR 코드를 스캔하세요.
  3. 경로 편집이 끝나면 경로 탐색을 누르세요.
  4. 재탐색된 경로를 선택한 후 안내 시작을 누르세요.

경유지 삭제하기

경유지로 설정된 장소를 경로에서 삭제할 수 있습니다.

  1. 검색/경로 화면에서 경로 편집을 누르세요.
  2. 경유지 중 경로에서 삭제할 장소를 선택한 후 삭제를 누르세요.
  3. 삭제한 장소를 제외하고 경로를 다시 탐색하려면 경로 탐색을 누르세요.
  4. 재탐색된 경로를 선택한 후 안내 시작을 누르세요.

경로 탐색 옵션 변경하기

경로를 탐색하는 옵션을 변경해 경로를 다시 탐색할 수 있습니다.

  1. 검색/경로 화면에서 경로 탐색 옵션을 누르세요.
  2. 원하는 경로를 선택하거나 경로 탐색 옵션을 누르세요.

    옵션 A(커넥티드 경로 탐색 화면)

    옵션 B(단말 경로 탐색 화면)

  3. 경로 탐색 화면에서 원하는 탐색 옵션의 적용 우선 순위를 변경하세요.
    • 화면에서 항목 오른쪽의 을 길게 누른 후 원하는 위치로 드래그하면 옵션 적용 우선 순위를 변경할 수 있습니다.
    • 탐색 옵션 화면에서 시스템의 웹 매뉴얼로 연결하려면 ▶ 웹 매뉴얼을 누른 후 QR 코드를 스캔하세요.
    • 차종 및 사양에 따라 선택할 수 있는 옵션이 다를 수 있습니다.

특정 구간 피하기

상습 정체 구간이나 공사 중인 구간 등 특정 도로를 제외한 경로로 다시 탐색할 수 있습니다.

  1. 검색/경로 화면에서 전체 경로를 누르세요.
  2. 경로 정보 / 회피를 누르세요.
  3. 피하려는 도로를 선택한 후 재탐색을 누르세요.
  4. 재탐색된 경로를 선택한 후 안내 시작을 누르세요.
  • 주행 중 주변에 공사 중인 곳이 있으면 해당 구간 제외 여부를 묻는 알림창이 나타납니다. 해당 구간을 피하려면 예를 눌러 경로를 재탐색하세요.
