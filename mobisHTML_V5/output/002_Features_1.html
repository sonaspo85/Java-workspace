유용한 기능

사용자 초기 설정하기

여러 명이 차량을 사용할 때 운전자별로 기본 사용 환경을 설정할 수 있습니다.

사용자 초기 설정에서는 화면에 표시될 사용자 이름 및 언어 등을 설정합니다.

  1. 안전 경고문이 나타나면 내용을 읽고, 설정하려는 운전자 이미지를 선택한 후 동의함을 누르세요.
    • 게스트로 로그인하면 프로필을 설정할 수 없습니다. 프로필을 설정하여 사용하려면 다른 사용자를 선택하세요.
    • 사용자는 최대 2명까지 등록할 수 있습니다.
  2. 사용자 초기 설정 화면이 나타나면 언어/Language를 누르세요.
    • Genesis Connected Services 계정을 연동하려면 시작을 누르세요. 해당 기능을 이용하려면 Genesis Connected Services에 가입하세요. ▶ '서비스 가입하기' 참고
    • 사용 환경을 변경하지 않고 기본 값으로 사용하려면 나중에를 누르세요.
  3. 시스템 화면에 표시할 언어를 설정한 후 다음을 누르세요.
  4. 운전자 이름을 설정하려면 이름 변경을 누르세요.

    이름 변경 화면이 표시됩니다.

    • 운전자 이름을 변경하지 않고 초기 설정을 진행하려면 다음을 누르세요.
  5. 사용자 초기 설정 완료 화면이 나타나면 확인을 누르세요.

    홈 화면으로 이동합니다.

    • 등록된 사용자의 세부 정보 및 사용 환경을 설정하려면 프로필 설정을 누르세요.
