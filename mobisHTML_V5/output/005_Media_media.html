미디어

미디어 사용하기

USB 모드로 USB 저장 장치나 MP3 플레이어 등 휴대용 기기에 저장된 음악/동영상 파일을 재생할 수 있으며, 블루투스 기능을 지원하는 블루투스 기기를 연결해 블루투스 음악을 재생할 수 있습니다.

USB로 음악 듣기

  1. 차량의 USB 단자에 음악을 재생할 미디어 기기를 연결하세요.
    • 재생하는 미디어 종류에 따라 재생이 바로 시작될 수 있습니다.
    • 사용할 수 있는 USB 저장 장치 및 파일 형식을 확인하세요. ▶  'USB 모드' 참고
    • 차종 및 사양에 따라 버튼이나 USB 단자의 형상 및 배열이 다를 수 있습니다.
  2. 다음 중 원하는 방법을 선택해 실행하세요.
    • •  홈 화면에서 방송/미디어 정보 위젯을 누르거나 전체 메뉴 화면에서 미디어를 누르세요.
    • •  조작 패널에서 [MEDIA] 버튼을 누르세요.
    • 여러 개의 미디어가 연결되어 있다면 [RADIO] 또는 [MEDIA] 버튼을 한 번 더 누른 후 방송/미디어 선택 창에서 원하는 미디어를 누르세요.

USB 음악 화면

  1. 이전 단계 화면으로 이동합니다.
  2. 파일 목록 화면으로 이동합니다.
  3. 현재 파일 번호 및 재생 파일 수
  4. 재생 중인 곡 정보. 아티스트나 앨범 정보를 누르면 현재 재생 중인 곡이 있는 아티스트 또는 앨범 목록으로 이동합니다.
  5. 반복 재생 모드를 변경합니다.
  6. 랜덤 재생을 켜거나 끕니다.
  7. 현재 곡 숨김
  8. 현재 파일을 처음부터 다시 시작하거나 이전 파일을 재생합니다.
  9. 재생을 일시 정지하거나 다시 시작합니다.
  10. 다음 파일을 재생합니다.
  11. 재생 중인 곡의 총 시간과 현재 재생 위치를 확인할 수 있습니다.
  12. 관련된 메뉴 목록이 표시됩니다.
    • •  화면 끄기: 화면을 끌 수 있습니다. 화면을 다시 켜려면 화면을 누르거나 전원 버튼을 짧게 누르세요.
    • •  미디어 선택: 방송/미디어 선택 창을 실행할 수 있습니다.
    • •  현재 아티스트의 곡: 재생 중인 곡이 있는 아티스트 목록으로 이동합니다.
    • •  현재 앨범의 곡: 재생 중인 곡이 있는 앨범 목록으로 이동합니다.
    • •  사운드 설정: 시스템 사운드를 설정할 수 있는 화면으로 이동합니다.
    • •  웹 매뉴얼 > USB 음악: 시스템의 웹 매뉴얼로 연결 가능한 QR 코드를 확인할 수 있습니다.
    • •  분할 화면: 분할 화면 모드를 켜거나 끌 수 있습니다.
  • 앰프 사양에 따라 화면에 음장 효과 설정 버튼이 표시될 수 있습니다. ▶ '음장 효과 조절하기(사양 적용 시)' 참고
  • 을 누르면 현재 재생 중인 곡을 즐겨찾기에 추가할 수 있습니다.
    • Genesis Connected Services에 가입한 경우에만 재생 화면에 해당 아이콘이 표시됩니다.
    • 재생 중인 곡에 곡명과 아티스트 정보가 있는 경우에만 재생 화면에 해당 아이콘이 표시됩니다.
    • 즐겨찾기 한 곡은 USB 음악 목록의 좋아요 카테고리에서 확인할 수 있습니다. 좋아요 카테고리에는 연결된 USB 저장장치의 음악 파일과 정보가 동일한 곡만 표시됩니다.

USB 음악 일시 정지/다시 재생하기

USB 음악 재생 중 음악을 잠시 멈추거나 정지된 음악을 다시 재생할 수 있습니다.

USB 음악 화면에서 또는 을 누르세요.

USB 음악 재생 중 빨리 감기

USB 음악 화면에서 을 길게 누르세요.

  • 스티어링 휠에서 이동 레버를 사용해 해당 기능을 실행할 수도 있습니다.

USB 음악 재생 중 현재 파일 다시 시작하기

USB 음악 재생 중 현재 파일을 처음부터 다시 재생할 수 있습니다.

USB 음악 화면에서 재생 3초 이후에 을 누르세요.

  • 스티어링 휠에서 이동 레버를 사용해 해당 기능을 실행할 수도 있습니다.

USB 음악 재생 중 이전 파일 재생하기

USB 음악 재생 중 이전 파일을 재생할 수 있습니다.

USB 음악 화면에서 재생 3초 이내에 을 누르세요.

  • 재생이 시작되고 3초 이후에는 을 두 번 눌러 이전 음악을 재생할 수 있습니다.
  • 스티어링 휠에서 이동 레버를 사용해 해당 기능을 실행할 수도 있습니다.
  • 조작 패널에서 탐색 노브를 사용해 원하는 음악을 검색할 수 있습니다. 검색 중 5초 이내에 노브를 누르면 음악이 재생되며, 5초 동안 동작이 없으면 검색이 중단되어 현재 재생 중인 파일 정보가 표시됩니다.

USB 음악 재생 중 다음 파일 재생하기

USB 음악 화면에서 을 누르세요.

  • 스티어링 휠에서 이동 레버를 사용해 해당 기능을 실행할 수도 있습니다.
  • 조작 패널에서 탐색 노브를 사용해 원하는 음악을 검색할 수 있습니다. 검색 중 5초 이내에 노브를 누르면 음악이 재생되며, 5초 동안 동작이 없으면 검색이 중단되어 현재 재생 중인 파일 정보가 표시됩니다.

USB 음악 반복 재생 모드 변경하기

USB 음악 재생 중 한 곡 반복, 폴더 반복, 전체 반복으로 반복 재생 모드를 변경할 수 있습니다.

USB 음악 화면에서 을 반복해서 누르세요.

누를 때마다 반복 재생 모드가 변경되며, 해당 모드 아이콘이 화면에 표시됩니다.

USB 음악 랜덤 재생 켜기/끄기

USB 음악 재생 중 랜덤 재생 모드를 켜거나 끌 수 있습니다.

USB 음악 화면에서 을 반복해서 누르세요.

누를 때마다 랜덤 재생 모드가 켜지거나 꺼집니다. 랜덤 재생 모드가 켜지면 전체 파일 순서와 상관없이 임의 순서로 파일이 재생됩니다.

USB 음악 목록 화면

USB 음악 화면에서 목록을 누르면 목록 화면에서 원하는 음악을 검색하여 재생할 수 있습니다.

  1. 이전 단계 화면으로 이동합니다.
  2. 상위 폴더/카테고리로 이동합니다.
  3. 카테고리별로 음악 파일을 확인할 수 있습니다. 아티스트 또는 앨범 카테고리에서는 원하는 아티스트나 앨범을 여러 개 선택해 재생할 수 있습니다.
    • Genesis Connected Services에 가입한 경우에만 좋아요 카테고리를 사용할 수 있습니다.
  4. 현재 카테고리의 모든 파일을 재생합니다.
  5. 재생 화면으로 이동합니다.
  6. 현재 카테고리 내의 모든 하위 폴더에 포함된 파일을 재생합니다.
  7. 관련된 메뉴 목록이 표시됩니다.
    • •  화면 끄기: 화면을 끌 수 있습니다. 화면을 다시 켜려면 화면을 누르거나 전원 버튼을 짧게 누르세요.
    • •  미디어 선택: 방송/미디어 선택 창을 실행할 수 있습니다.
    • •  현재 곡 이동: 목록에서 이동하며 파일을 검색하는 중 현재 재생 중인 파일이 있는 위치로 복귀할 수 있습니다.
    • •  분할 화면: 분할 화면 모드를 켜거나 끌 수 있습니다.

주의

  • •  반드시 시동을 건 상태에서 USB 기기를 연결하세요. USB 기기를 연결한 후에 시동을 걸면 해당 USB 기기가 고장 날 수 있습니다.
  • •  USB 기기를 연결하거나 분리할 때 정전기에 주의하세요. 정전기로 인해 시스템이 고장 날 수 있습니다.
  • •  차량의 USB 단자에 신체나 외부 물체가 닿지 않도록 주의하세요. 사고나 시스템 고장의 원인이 될 수 있습니다.
  • •  USB 커넥터를 짧은 시간에 연결하고 분리하는 동작을 반복하지 마세요. 기기 동작에 오류가 발생할 수 있으며 시스템이 고장 날 수 있습니다.
  • •  USB 기기를 미디어 파일 재생 이외의 용도로 사용하지 마세요. 충전이나 발열 기능이 있는 USB 액세서리를 연결하면 시스템의 성능이 저하되거나 고장의 원인이 될 수 있습니다.
  • USB 저장 장치를 연결할 때에는 별도의 연장 케이블을 사용하지 말고 차량의 USB 단자에 직접 연결하세요. USB 허브 또는 연장 케이블로 연결하면 장치가 인식되지 않을 수 있습니다.
  • USB 커넥터를 차량의 USB 단자 끝까지 밀어 넣으세요. 커넥터가 완전히 꽂히지 않으면 통신에 문제가 생길 수 있습니다.
  • USB 기기를 시스템에서 분리할 때 잡음이 발생할 수 있습니다.
  • 표준 형식으로 인코딩된 파일만 재생할 수 있습니다.
  • 다음과 같은 USB 기기는 인식되지 않거나 정상적으로 동작하지 않을 수 있습니다.
    • 암호화된 MP3 플레이어
    • 이동식 디스크로 인식되지 않는 USB 기기
  • USB 기기의 상태에 따라 인식되지 않을 수 있습니다.
  • 일부 USB 기기는 호환성 문제로 지원되지 않을 수 있습니다.
  • USB 기기의 종류, 용량, 저장된 파일의 형식 등에 따라 USB 기기 인식 시간이 길어질 수 있습니다.
  • USB 사양에 따라 일부 기기에서는 USB 연결을 통한 충전이 지원되지 않을 수 있습니다.
  • 파일 숨김 기능은 exFAT, FAT16/32, NTFS 형식의 USB 저장 장치에서만 사용할 수 있습니다. 숨긴 파일은 'Hidden_' 폴더에서 확인할 수 있습니다.

  • 재생이 시작되고 3초 이후에는 을 두 번 눌러 이전 동영상을 재생할 수 있습니다.
  • 스티어링 휠에서 이동 레버를 사용해 해당 기능을 실행할 수도 있습니다.

USB 동영상 화면에서 을 반복해서 누르세요.

누를 때마다 반복 재생 모드가 변경되며, 해당 모드 아이콘이 화면에 표시됩니다.

USB 동영상 목록 화면

USB 동영상 화면에서 목록을 누르면 목록 화면에서 원하는 동영상을 검색하여 재생할 수 있습니다.

  1. 이전 단계 화면으로 이동합니다.
  2. 상위 폴더로 이동합니다.
  3. 현재 카테고리의 모든 파일을 재생합니다.
  4. 재생 화면으로 이동합니다.
  5. 현재 카테고리 내의 모든 하위 폴더에 포함된 파일을 재생합니다.
  6. 관련된 메뉴 목록이 표시됩니다.
    • •  화면 끄기: 화면을 끌 수 있습니다. 화면을 다시 켜려면 화면을 누르거나 전원 버튼을 짧게 누르세요.
    • •  미디어 선택: 방송/미디어 선택 창을 실행할 수 있습니다.
    • •  현재 파일 이동: 목록에서 이동하며 파일을 검색하는 중 현재 재생 중인 파일이 있는 위치로 복귀할 수 있습니다.
    • •  분할 화면: 분할 화면 모드를 켜거나 끌 수 있습니다.

주의

  • •  반드시 시동을 건 상태에서 USB 기기를 연결하세요. USB 기기를 연결한 후에 시동을 걸면 해당 USB 기기가 고장 날 수 있습니다.
  • •  USB 기기를 연결하거나 분리할 때 정전기에 주의하세요. 정전기로 인해 시스템이 고장 날 수 있습니다.
  • •  차량의 USB 단자에 신체나 외부 물체가 닿지 않도록 주의하세요. 사고나 시스템 고장의 원인이 될 수 있습니다.
  • •  USB 커넥터를 짧은 시간에 연결하고 분리하는 동작을 반복하지 마세요. 기기 동작에 오류가 발생할 수 있으며 시스템이 고장 날 수 있습니다.
  • •  USB 기기를 미디어 파일 재생 이외의 용도로 사용하지 마세요. 충전이나 발열 기능이 있는 USB 액세서리를 연결하면 시스템의 성능이 저하되거나 고장의 원인이 될 수 있습니다.
  • USB 저장 장치를 연결할 때에는 별도의 연장 케이블을 사용하지 말고 차량의 USB 단자에 직접 연결하세요. USB 허브 또는 연장 케이블로 연결하면 장치가 인식되지 않을 수 있습니다.
  • USB 커넥터를 차량의 USB 단자 끝까지 밀어 넣으세요. 커넥터가 완전히 꽂히지 않으면 통신에 문제가 생길 수 있습니다.
  • USB 기기를 시스템에서 분리할 때 잡음이 발생할 수 있습니다.
  • 표준 형식으로 인코딩된 파일만 재생할 수 있습니다.
  • 다음과 같은 USB 기기는 인식되지 않거나 정상적으로 동작하지 않을 수 있습니다.
    • 암호화된 MP3 플레이어
    • 이동식 디스크로 인식되지 않는 USB 기기
  • USB 기기의 상태에 따라 인식되지 않을 수 있습니다.
  • 일부 USB 기기는 호환성 문제로 지원되지 않을 수 있습니다.
  • USB 기기의 종류, 용량, 저장된 파일의 형식 등에 따라 USB 기기 인식 시간이 길어질 수 있습니다.
  • USB 사양에 따라 일부 기기에서는 USB 연결을 통한 충전이 지원되지 않을 수 있습니다.

블루투스로 음악 듣기

블루투스 기기를 사용해 오디오를 재생하려면 블루투스 기능을 지원하는 휴대 전화나 MP3 플레이어 등 다른 기기를 시스템에 연결해야 합니다.

블루투스 연결에 관한 자세한 내용은 다음을 참고하세요. ▶  '블루투스 기기 연결하기' 참고

  • 연결된 블루투스 기기나 휴대 전화에 따라, 또는 사용하는 뮤직 플레이어 앱에 따라 조작 방법 및 화면이 다를 수 있습니다.
  • 사용하는 뮤직 플레이어 앱에 따라 스트리밍 기능이 지원되지 않을 수 있습니다.
  • 연결된 블루투스 기기 또는 휴대 전화에 따라 일부 기능이 지원되지 않을 수 있습니다.
  • 미디어 기기나 휴대 전화를 블루투스와 USB 케이블로 동시에 연결하면 기기나 시스템이 올바르게 작동하지 않을 수 있습니다. 블루투스 연결을 이용하려면 기기에 연결된 케이블을 모두 분리하세요.

블루투스 오디오 화면

블루투스로 연결된 오디오 기기에서 음악 파일을 재생하면 다음과 같은 음악 플레이어가 실행됩니다.

  1. 이전 단계 화면으로 이동합니다.
  2. 재생 중인 곡 정보
  3. 현재 곡을 처음부터 다시 시작하거나 이전 곡을 재생합니다.
  4. 재생을 일시 정지하거나 다시 시작합니다.
  5. 다음 곡을 재생합니다.
  6. 재생 중인 곡의 총 시간과 현재 재생 위치를 확인할 수 있습니다.
  7. 연결된 기기에서 재생 중인 곡의 온라인 앨범 아트가 표시됩니다. 온라인 앨범 아트는 카카오 i 서비스로 검색되어 제공됩니다.
    • 카카오 i 서비스로 제공되는 온라인 앨범 아트는 연결된 기기에 표시되는 앨범 아트와 다를 수 있습니다.
    • 온라인 앨범 아트 검색에는 시간이 소요되어 곡 시작과 앨범 아트 표시 시점에 차이가 있을 수 있습니다.
    • 카카오 i 서비스를 이용하려면 Genesis Connected Services에 가입하세요. ▶  '서비스 가입하기' 참고
  8. 재생 파일 수
  9. 다른 블루투스 기기를 찾아 연결할 수 있습니다.
  10. 곡 목록 화면으로 이동합니다.
    • 블루투스 오디오 목록은 연결된 기기나 앱에 따라 표시되지 않을 수 있습니다.
  11. 관련된 메뉴 목록이 표시됩니다.
    • •  화면 끄기: 화면을 끌 수 있습니다. 화면을 다시 켜려면 화면을 누르거나 전원 버튼을 짧게 누르세요.
    • •  미디어 선택: 방송/미디어 선택 창을 실행할 수 있습니다.
    • •  기기 변경: 다른 블루투스 기기를 찾아 연결할 수 있습니다.
    • •  사운드 설정: 시스템 사운드를 설정할 수 있는 화면으로 이동합니다.
    • •  웹 매뉴얼 > 블루투스 오디오: 시스템의 웹 매뉴얼로 연결 가능한 QR 코드를 확인할 수 있습니다.
    • •  분할 화면: 분할 화면 모드를 켜거나 끌 수 있습니다.
  • 앰프 사양에 따라 화면에 음장 효과 설정 버튼이 표시될 수 있습니다. ▶ '음장 효과 조절하기(사양 적용 시)' 참고
  • 을 누르면 현재 재생 중인 곡을 즐겨찾기에 추가할 수 있습니다.
    • Genesis Connected Services에 가입한 경우에만 재생 화면에 해당 아이콘이 표시됩니다.
    • 재생 중인 곡에 곡명과 아티스트 정보가 있는 경우에만 재생 화면에 해당 아이콘이 표시됩니다.
    • 즐겨찾기 한 곡은 USB 음악 목록의 좋아요 카테고리에서 확인할 수 있습니다. 좋아요 카테고리에는 연결된 USB 저장장치의 음악 파일과 정보가 동일한 곡만 표시됩니다.

뮤직 스트리밍 서비스는 음원 컨텐츠 플랫폼의 스트리밍 서비스를 구독 중인 경우에 사용 가능합니다. 또한 Genesis Connected Services 앱에서 차량용 스트리밍 데이터 요금제를 별도 구매 후 사용 가능합니다.

  • Genesis Connected Services에서 지원하는 음원 컨텐츠 플랫폼을 확인하세요.
